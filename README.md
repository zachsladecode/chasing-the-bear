# Chasing the Bear

A tech blog about AI/ML and Software Engineering built with Astro and GitHub Pages.

Live at [chasingthebear.net](https://chasingthebear.net)

## Stack

- [Astro](https://astro.build) — static site framework
- GitHub Pages — hosting
- GitHub Actions — CI/CD
- Markdown/MDX — content authoring
